## How to use this `awesome‑ai‑lists` repo

- **Want to navigate the ecosystem?** Start with `josephmisiti/awesome-machine-learning` and `neomatrix369/awesome-ai-ml-dl` for the “mother‑ship” lists.  
- **Want prompts and evals?** Use the **prompt‑engineering, evaluation, and jailbreak / red‑team** section.  
- **Want safety/security?** Dive into the **AI safety, security, privacy, guardrails** category.  
- **Want infra/MLOps?** Use the **AI infrastructure, MLOps, LLMops, data / models** section.  
- **Want domain‑specific AI?** Pick the sub‑section (science, medicine, finance, etc.) that matches your domain.  

Pull requests welcome to:
- Fix broken links  
- Add a new “awesome‑style” list that fits one of these categories
